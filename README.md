vagrant-sunzi-standard
=======================

いろいろ詰め合わせ

手順書的なもの
------

個人的にいちいち環境を構築するのがめんどくさかったので、standard版として開発基盤的な環境を作ってみる
(とりあえずの暫定版でそのあといろいろアップデートする予定)


Installation
------

### mac

```
$ git clone 
$ vagrant up --provision
$ cd sunzi
$ sunzi deploy localhost:2222
```


### virutal box

```
$ cd dotfiles
$ sh bootstrap.sh
$ vim
$ :NerBundleInstall
```
